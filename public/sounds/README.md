# Sound Files

All sound files should be placed in this directory structure.
Each file should be a short MP3 (< 100KB for effects, < 5MB for music).

## Effects (`/sounds/effects/`)

| File | Duration | Description |
|------|----------|-------------|
--| `correct.mp3` | ~0.5s | Bright positive chime, happy tone |
--| `wrong.mp3` | ~0.5s | Soft "whoops" — gentle, not punitive |
--| `launch.mp3` | ~1.5s | Rocket ignition whoosh |
--| `landing.mp3` | ~2s | Satisfying thud + spacey fanfare |
--| `click.mp3` | ~0.1s | Soft UI tap sound |
| `streak-3.mp3` | ~0.8s | Ascending chimes × 3 |
| `streak-5.mp3` | ~1s | Turbo whoosh with bells |
| `streak-10.mp3` | ~1.5s | Epic fanfare burst |
| `badge.mp3` | ~1.5s | Achievement unlock sound (triumphant) |
| `countdown.mp3` | ~0.8s | Beep for countdown numbers |
| `star.mp3` | ~0.6s | Star sparkle ping |

## Music (`/sounds/music/`)

Background ambient music per galaxy. Loop-friendly (seamless loop points).

| File | Duration | Style |
|------|----------|-------|
| `galaxy-1.mp3` | 2-3 min | Upbeat space pop, lighthearted (Solar System) |
| `galaxy-2.mp3` | 2-3 min | Mysterious ambient, slightly darker (Outer Reaches) |
| `galaxy-3.mp3` | 2-3 min | Epic orchestral, cinematic (The Stars) |
| `galaxy-4.mp3` | 2-3 min | Deep cosmic, ethereal, transcendent (Deep Space) |

All tracks should be:
- Royalty-free or custom-created
- Loopable (intro plays once, then loop section repeats)
- Child-appropriate, no lyrics
- Volume-normalized at -14 LUFS
